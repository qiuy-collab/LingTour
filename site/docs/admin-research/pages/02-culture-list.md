# 文化列表页（/culture）后台数据调研

## 页面目标
按城市展示文化卡片，支持用户进入城市文化详情。

## 前端需要后台管理的数据
- 城市列表（`/public/cities`）
  - slug/name/regionLabel/heroImage/heroNarrative/tags/editorIntro
  - 可选：排序权重、推荐位标记

## 后台设计
- 模块：`城市管理`
  - 基本信息、标签、封面图、摘要、状态（draft/published）
- 模块：`城市排序`
  - 手工拖拽排序、按区域筛选排序
- 模块：`展示控制`
  - 上线时间、下线时间、SEO信息

## 后台用户流程
1. 内容编辑创建/更新城市。
2. 上传封面图并维护 narrative 与标签。
3. 在“城市排序”调整前台展示顺序。
4. 提交审核并发布。
