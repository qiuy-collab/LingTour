# 前端页面重构实施计划

## Context

用户希望将当前前端从"内容信息较繁琐"的展示方式，重构为更接近 Shopify 高端独立站的视觉与转化体验：少而精的内容层级、强视觉叙事、清晰 CTA、路线/口译/社区形成商业闭环。本次计划不直接修改代码，而是在现有代码结构基础上给出可执行的重构方案。

## 设计目标

1. **高端 Shopify 独立站风格**：大图、留白、精品卡片、明确转化入口，减少密集文字堆叠。
2. **首页从"介绍平台"转为"触发灵感"**：近期事件/节庆路线推荐 + 广东活动日历。
3. **路线页从"卡片列表"转为"地图选路线"**：省/地区/子区域地图，多路线 polyline，选中高亮，进入详情。
4. **路线详情从"长文滚动"转为"节点索引式体验"**：每个 stop 承载故事、安排、地点链接与地图索引。
5. **社区内容进入路线闭环**：用户可看 route posts/comments；已预订用户才能评论对应路线。
6. **口译页套餐化**：选择场景套餐 + 口译员等级/人员，生成组合价格并进入预约。

## 实施优先级

1. UI 设计系统定义与页面信息层级收敛。 ✅
2. 首页事件路线轮播 + 广东活动日历。 ✅
3. 路线列表页地图化。 ✅
4. 路线详情页节点索引 + 路线地图 + 安排详情。 ✅
5. 口译页套餐化 + booking 对接。 ✅
6. posts/comments 后端模型与 route community section。 ✅
7. 全站社区入口 `/posts` 与用户帖子页。 ✅
8. 响应式、空状态、加载状态、动效与性能 polish。 ✅
9. 最终全量测试与真实浏览器用户流程质量评估。 ✅
