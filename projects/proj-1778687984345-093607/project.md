# 项目：前端页面高端化重构 (Shopify 对标)

## 项目目标
将现有前端页面重构为具备 Shopify 独立站风格的高端设计，优化用户路径，增强地图交互性，并引入基于订单的社交分享机制。

## 核心功能
1.  **动态事件系统**：近期风俗节日轮播图 + 广东地区文化事件日历。
2.  **地图驱动路线系统**：
    -   大地图（省）-> 地区地图（市/县）逐级下钻。
    -   地图实时展示多条真实路线，支持高亮选择。
    -   路线详情页与地图联动，以节点为索引进行内容切换。
3.  **社交评论系统**：
    -   订单锁定评论（仅限购买过的路线）。
    -   节点级 Post 展示（用户可以在特定节点看到他人的故事和评论）。
4.  **精简口译服务页**：场景化套餐 + 口译员分级选择 + 动态组合计价。

## 里程碑
- [ ] ⬜ **Phase 1: 设计方案与视觉基调定义** (对标 Shopify 高端感)
- [ ] ⬜ **Phase 2: 首页 & 事件日历重构** (轮播与文化日历)
- [ ] ⬜ **Phase 3: 地图交互路线系统开发** (省市联动与路径展示)
- [ ] ⬜ **Phase 4: 路线详情页与社交功能集成** (节点索引、故事展示、Post 系统)
- [ ] ⬜ **Phase 5: 口译页精简与体验优化** (组合计价逻辑)
- [ ] ⬜ **Phase 6: 全局测试与性能调优**

## 工作方法
- **设计原则**：大留白、精致字体、高质量动效、响应式优先。
- **技术约束**：组件化开发，确保地图性能顺滑。

## 风险管理
- **地图兼容性**：复杂路线在小地图上的性能开销。
- **数据联动**：订单、路线节点与 Post 之间的权限与关联逻辑。
