# 长期记忆

## 2026-05-13 (项目初始化)
- **视觉目标**：明确对标 Shopify 独立站，风格定位为高端、简约、精致。
- **核心逻辑**：
    - 社交评论（Post）实行“订单锁定”机制，增强社区可信度。
    - 地图重构采用“下钻式”交互（省 -> 区 -> 路线）。
    - 路线详情页将地图节点作为内容索引（Node-based Navigation）。
- **页面需求**：
    - 加入风俗节日事件轮播及广东地区活动日历。
    - 口译页精简为：场景套餐 + 口译员等级 + 组合价格。
    - 全局 Post/Comment 入口，支持跨用户浏览。
