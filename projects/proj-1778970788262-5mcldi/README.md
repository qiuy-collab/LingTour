# LingTour Admin Web

这是凌云游 (LingTour) 项目的 Web 管理后台。主要用于对移动端或 C 端产品的各项核心数据进行运营和维护。

## 功能特性
- **Dashboard**: 查看运营核心数据
- **城市管理 (Cities)**: 目的地信息维护
- **路线管理 (Routes)**: 旅游路线编辑与上下架
- **商城管理 (Shop)**: 特产商品及周边上架与库存
- **翻译服务 (Interpreting)**: 导游/翻译的入驻审核与管理
- **用户管理 (Users)**: 注册用户状态控制
- **订单管理 (Orders)**: 路线、商城、翻译的订单处理

## 技术选型
- [Vue 3](https://vuejs.org/)
- [Vite](https://vitejs.dev/)
- [Element Plus](https://element-plus.org/)
- [TypeScript](https://www.typescriptlang.org/)
