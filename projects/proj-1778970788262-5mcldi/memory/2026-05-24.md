## 城市 Sections 编辑功能修复

### 根因
后端 `cities.service.ts` 的 `Object.assign(city, dto)` 把 sections 赋给 city 后，`save(City, city)` 因 Entity cascade 尝试 INSERT 无 id 的 sections，与数据库中已有 sections 冲突 → 500。

### 修复
- **后端**：在 save 前 `delete (city as any).sections`，阻止级联保存
- **前端**：CityEdit.vue 移除所有只读限制（alert、只读标签、禁用按钮、死代码）

### 验证
- curl 测试 edit/add/delete/reorder sections 全部 200
- 浏览器保存成功并持久化到后端
- vue-tsc 零错误 + vite build 1.30s
